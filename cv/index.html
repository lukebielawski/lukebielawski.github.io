  • 01.2024 - 05.2024
    Illinois Mathematics Lab: Linear Poisson Structures and their Cohomology at the University of Illinois
    Poisson structures are special geometric objects, originally arising out of classical mechanics. One particular class of them arises out of any Lie algebra, called a linear Poisson structure. In this project, we calculated the Poisson cohomlogy of different Linear Poisson structures (with power series solutions), which gave insight to the geometry of these structures. I also created Python and Sage code to be able to automatically preform these calculations.
    • Calculated Cohomology groups for various linear Poisson structures via power series
    • Wrote Python code to automatically calculate Cohomology classes with finite degree power series
    • Exhibited findings at Undergraduate Research Symposium and IML Poster Session
